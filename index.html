Artifact Overview

The primary artifact included in this portfolio is the CS340 Grazioso Salvare Dashboard, a Python-based dashboard application that uses Dash and MongoDB to retrieve, filter, and visualize animal shelter data. This artifact was enhanced across three capstone categories: software design and engineering, algorithms and data structures, and databases.

Enhancement One: Software Design and Engineering

This enhancement improved the structure of the application by refactoring the filtering and query logic into modular backend functions. The goal was to improve readability, maintainability, and separation of concerns.
